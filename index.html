입력 방식과 환경을 여기서 한 번에 관리해요.

입력 방식 안내

  • 키보드: 방향키 또는 WASD
  • 모바일: 화면 D-pad 터치
  • 음성: “앞으로”, “뒤로”, “왼쪽”, “오른쪽”, “먹기”
  • DotPad movement: PanningLeft/Right and LPF1/RPF4 / 좌·우·상·하 이동
  • DotPad combos (KeyFunction12/34 etc.): collect or interact / 먹기·상호작용
  • DotPad: 패닝 좌·우 = 좌우 이동, F1·F2 = 앞·뒤 이동, F3 = 먹기, F4 = 촉각 지도, Pan All = 주변 안내
  • 정보 듣기: 1·2·3·4 키 또는 화면 버튼 — 위치 · 주변 · 미션 · 촉각 지도
